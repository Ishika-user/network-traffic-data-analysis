# Network Traffic Data Analysis

This repository contains my CC5067NT Smart Data Discovery coursework based on network traffic data analysis using Jupyter Notebook.

## Project Overview

The aim of this project is to analyse network traffic data and understand important patterns in the dataset. The project includes data understanding, dataset structure, data cleaning, exploratory data analysis, and interpretation of network traffic features.

Network Traffic Data Analysis

This project was completed as part of the CC5067NT Smart Data Discovery coursework. It demonstrates basic data analysis skills using Python and Jupyter Notebook.
